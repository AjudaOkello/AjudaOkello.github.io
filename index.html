About Me

I am an Electrical and Electronic Engineering student at Griffith University’s Gold Coast campus with a strong passion for programming, electronics, and computer architecture. I particularly enjoy courses involving circuit simulation and practical implementation on breadboards, as they allow me to apply theoretical concepts in a hands on environment.

My interests focus on hardware components and embedded programming, particularly in circuit design, embedded systems, control systems, and electronic analysis. I enjoy understanding how hardware and software interact at a low level, especially through practical implementation, simulation, and system level design.

I have experience with engineering and simulation tools, including CircuitLab, EasyEDA, SPICE, and PUFF, and have developed programming skills in both C and Python, with C as my preferred language.

I am especially interested in computer electronic design, including modern motherboards and hardware systems, as well as aeronautical and aerospace technologies, due to the large number of advanced electronic systems used in modern aircraft and space applications.

Projects

Book Library Management System

This project is a console based Book Library Management System developed using the C programming language. The system was designed to manage books and users through a menu driven interface that allows users to add books, view books, delete books, borrow books, return books, and manage library users. This project helped me strengthen my understanding of programming logic, functions, structures, and menu based application design.

PWM DC Motor Speed Control

This project focuses on the design and implementation of a PWM DC Motor Speed Control System. The system is designed to control motor speed from 0% to 100% duty cycle using a triangle wave generator, comparator circuit, isolation circuit, and drive circuit. It also includes a transformer, rectifier, and filter stage for power conversion. This project improved my understanding of PWM control, motor drive circuits, circuit simulation, and practical electronics design.
